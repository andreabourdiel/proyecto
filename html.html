HTML

Aprende sobre el lenguaje marcado para estructurar páginas web.

¿Qué es HTML?

HTML significa HyperText Markup Language. Es el lenguaje estándar para crear páginas web y aplicaciones web.

Consiste en una serie de elementos que están compuestas por la etiqueta de apertura, el contenido, y la etiqueta de cierre.

Las etiquetas a su vez se componen del nombre del elemento encerrado por paréntesis angulares (<>), excepto que en la etiqueta de cierre se le agrega una barra de cierre (/) antes del nombre de la etiqueta.

Cada uno de estos elemento cumplen con una función específica, el más importante es el <html> porque le indica a la máquina que se trata de un archivo en lenguaje html. Otras etiquetas importantes son las que definen la sección del documento, estas son:

  • <head>: Aquí se pondrá información general acerca del documento (nombre, idioma, escala, etc.)

  • <body>: Aquí se pondrá el contenido de la página (texto, imagenes, tablas, hipervínculos, etc.)

  • <footer>: Aquí se pondrá el nombre del autor o los autores.

Otra de las etiquetas más importantes son las que se usan para poner títulos y subtitulos. Consiste en una h seguida de un número del 1 al 6, siendo el 1 el más grande y el 6 el más chico.

Para poner texto se usa la etiqueta <p> que significa párrafo. Para poner imágenes usamos la etiqueta <img>, en dónde pondremos el nombre de la imagen (src), una descripción (alt), y también podremos especificar su altura (height) y/o ancho (width)

También se puede poner hipervínculos con la etiqueta <a>, pondremos la url de la página o video y el texto que queremos clickear.
