CSS

Descubre cómo usar CSS para darle estilo a tus páginas web.

¿Qué es CSS?

CSS significa Cascading Style Sheets. Es un lenguaje gráfico para definir la presentación de documentos HTML.

Consiste en un serie de etiquetas que se conforman por el nombre del elemento y entre llaves los atributos que queremos que tengan.

Unos de los atributos más comunes son:

  • color: Establece el color de un elemento.

  • background-color: Establece el color de fondo.

  • text-size: Establece el tamaño de letra.

  • text-font: Establece la fuente de la letra.

  • text-align: Establece la alineación horizontal del texto.

  • margin: Establece los márgenes de un elemento.

  • padding: Establece el espacio o área alrededor del contenido de un elemento.

  • height: Establece la altura de un elemento.

  • width: Establece el ancho de un elemento.

Hay todavía más etiquetas, pero estás son suficientes para empezar. Si quieres descubrir más, tan solo busca etiquetas CSS en internet.
